# Syspago Project

Syspago is a web application designed for mobile point-of-sale (TPV) functionalities. This project allows users to manage various services and transactions efficiently.

## Project Structure

```
syspago
├── index.html         # Main structure of the web application
├── css
│   └── styles.css     # Custom styles for the application
├── js
│   └── app.js         # JavaScript logic for dynamic content and interactions
├── package.json       # Configuration file for npm
└── README.md          # Documentation for the project
```

## Features

- **Dynamic Header**: The header includes a logo and title that can be updated dynamically.
- **Main Content Area**: This area displays different views based on user interactions.
- **Commission Modal**: A modal for applying commissions with options for percentage or fixed amounts.
- **Services Menu**: A dedicated section for various services including:
  - Agua y drenaje
  - Catalogos
  - Gas y electricidad
  - Gobiernos
  - Loteria
  - Movilidad
  - Servicios digitales
  - TV e internet
- **Responsive Design**: The application is designed to be mobile-friendly using Tailwind CSS.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd syspago
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

- Open `index.html` in a web browser to view the application.
- Use the bottom navigation bar to switch between different views.
- Access the services menu to explore available services.
